# tic-tac-toe

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)

Репозиторий: **https://github.com/akarakuts/tic-tac-toe**

```bash
git clone https://github.com/akarakuts/tic-tac-toe.git
cd tic-tac-toe
```

Нативная игра **«Крестики-нолики»** для **macOS** (линия из *n* подряд на квадратном поле), на **SpriteKit** и **Swift**. Два игрока за одним экраном или один человек против компьютера. Интерфейс поддерживает размер поля, условие победы, разблокируемые темы оформления доски и сохраняемую статистику против ИИ.

Полная версия на английском: [`README.md`](README.md).

## Возможности

- **Размер поля** — от **3×3** до **6×6**, выбор в левой колонке настроек.
- **Условие победы** — длина линии от **3** до **6**; значения больше стороны поля недоступны (должно быть ≤ размера доски). Учитываются строки, столбцы и обе диагонали.
- **Два игрока** — поочерёдные ходы на одном поле (первыми ходят ✕).
- **Против компьютера** — минимакс с альфа-бета отсечением; можно выбрать сторону — крестики (✕) или нолики (○).
  - На **3×3** при трёх в ряд и сложности **«Сложно»** перебор достаточно глубокий для по сути оптимальной игры. У **«Легко»** и **«Средне»** глубина всегда ограничена (ИИ слабее и «человечнее»).
  - На **больших полях** поиск ограничен по глубине с позиционной эвристикой; по-прежнему учитываются немедленный выигрыш и (как правило) блок. **«Легко»** иногда не блокирует угрозу или ходит наугад; **«Средне»** иногда выбирает почти лучший ход.
- **Сложность ИИ** — **Легко**, **Средне**, **Сложно** (всегда видна в левой колонке). В режиме **«Два игрока»** элементы остаются на панели, но **неактивны**; в режиме **против компьютера** — активны. Выбор сохраняется между запусками.
- **Темы доски** — **Классика** и три палитры (**Аврора**, **Роща**, **Закат**) поверх светлой/тёмной темы системы и акцентного цвета (`BoardAppearance.swift`).
  - **Классика** доступна всегда.
  - Разблокировки считаются только за партии **против компьютера**: **Аврора** — после **первой** вашей победы над ИИ; **Роща** — после **5** накопительных побед над ИИ; **Закат** — когда **лучшая серия** побед достигла **3** (текущая серия обнуляется при поражении или ничьей). Короткая легенда условий показана **под игровым полем**.
- **Статистика** — строка внизу панели: **П–П–Н** против ИИ, **текущая серия** и **лучшая серия**; обновляется по завершении раунда против ИИ. Хранится вместе с прогрессом в **UserDefaults**.
- **Звук** — необязательные звуки хода / победы / ошибки (`GameSoundFX`); переключатель **звук вкл./выкл.** сохраняется.
- **Вёрстка** — элементы управления в **левой колонке** (подписи над группами пилюль); **сетка** занимает оставшееся место. Панель и HUD масштабируются вместе с окном, а легенда условий разблокировки тем находится **под игровым полем**.
- **Локализация** — строки в **`Localizable.xcstrings`** (String Catalog): **40 идентификаторов локали** в духе типичных настроек macOS (включая региональные варианты вроде `en-GB`, `es-419`, `fr-CA`). Исходные формулировки правятся в **`tic-tac-toe/Scripts/write_bundle.py`** и **`bundle_strings.json`**, каталог пересобирается скриптами ниже; активный язык следует **системным настройкам**.
- **Оформление** — анимация ходов, подсветка выигрышной линии, лёгкая обратная связь при ничьей и исходе партии.

## Требования

- macOS (целевая версия и SDK — в проекте Xcode)
- Xcode со Swift 5+ и поддержкой String Catalog

## Сборка и запуск

1. Откройте `tic-tac-toe.xcodeproj` в Xcode.
2. Выберите схему **tic-tac-toe** и **My Mac**.
3. Запуск (**⌘R**).

Сборка из командной строки:

```bash
xcodebuild -scheme "tic-tac-toe" -destination 'platform=macOS' build
```

## Пересборка локализованных строк

После правок переводов в `tic-tac-toe/Scripts/write_bundle.py` (или в сгенерированном `bundle_strings.json`) обновите каталог строк из каталога **Scripts**:

```bash
cd tic-tac-toe/Scripts
python3 write_bundle.py          # обновляет bundle_strings.json из write_bundle.py
python3 generate_localizable.py  # записывает ../Localizable.xcstrings
```

## Тесты

Юнит-тесты в таргете **tic-tac-toeTests** (Swift Testing). Пример:

```bash
xcodebuild -scheme "tic-tac-toe" -destination 'platform=macOS' -only-testing:tic-tac-toeTests test
```

Правила поля покрыты тестами `GameModel`; поведение ИИ (в том числе не только 3×3) — `TicTacToeAITests`.

## Структура проекта

| Путь | Назначение |
|------|------------|
| `tic-tac-toe/GameModel.swift` | Состояние доски, ходы, поиск выигрышной линии |
| `tic-tac-toe/TicTacToeAI.swift` | ИИ на минимаксе, ограничения сложности, тактика выигрыша/блока |
| `tic-tac-toe/GameScene.swift` | Сцена SpriteKit: раскладка, сетка, ввод, HUD, темы и статистика |
| `tic-tac-toe/GameProgress.swift` | Сохраняемая статистика, серии, разблокировки, сложность, звук |
| `tic-tac-toe/BoardAppearance.swift` | Стили доски и палитры тем |
| `tic-tac-toe/GameSoundFX.swift` | Звуковые эффекты интерфейса |
| `tic-tac-toe/ViewController.swift` | Хостинг сцены в `SKView`, размеры и режим масштаба |
| `tic-tac-toe/L10n.swift` | Доступ к локализованным строкам |
| `tic-tac-toe/Localizable.xcstrings` | String Catalog (генерируется; см. Scripts) |
| `tic-tac-toe/Scripts/write_bundle.py` | Редактирование переводов → `bundle_strings.json` |
| `tic-tac-toe/Scripts/generate_localizable.py` | Сборка `Localizable.xcstrings` из набора строк |
| `tic-tac-toe/Scripts/strings_bundle.py` | Загрузка `bundle_strings.json` для генератора |

## Релизы и установка из GitHub

При пуше **тега** GitHub Actions собирает конфигурацию **Release**, создаёт [Release](https://github.com/akarakuts/tic-tac-toe/releases) с автогенерацией заметок и прикладывает образ **`tic-tac-toe-<тег>-macOS.dmg`** (внутри — `tic-tac-toe.app` и ярлык **Applications**). Сборка без подписи разработчика: при первом запуске может потребоваться открыть через контекстное меню («Открыть»).
